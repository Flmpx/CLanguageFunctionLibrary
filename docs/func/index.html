📋 List 链表

双向链表 (DList) - 支持头插、尾插、指定位置插入、删除、反转等操作

1.1 DList (双向链表)

🔢

1.1.1 DList_S - 单一数据类型

存储单一类型数据（如 int、string），需要提供数据操作集
🔣

1.1.2 DList_M - 多种数据类型

可存储任意类型数据，每个数据自带操作函数

🗺️ Map 映射表

键值对数据结构，支持 key-value 存储与查找

1.2 ChainMap (链表法哈希映射)

采用链表法解决哈希冲突，适合数据量较大且分布不均匀的场景

🔢

1.2.1 ChainMap_S - 单一数据类型

Key 和 Value 都为单一类型
🔣

1.2.2 ChainMap_M - 多种数据类型

Key 和 Value 可为任意类型

1.3 OAMap (开放定址法哈希映射)

采用开放定址法解决哈希冲突，适合数据量较小且分布均匀的场景

🔢

1.3.1 OAMap_S - 单一数据类型

Key 和 Value 都为单一类型
🔣

1.3.2 OAMap_M - 多种数据类型

Key 和 Value 可为任意类型
